# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Настройка MySQL

Создайте базу данных:

```sql
CREATE DATABASE portals_bot CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

## 3. Получение ключей

### Токен бота
1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Сохраните токен

### API ID и API Hash
1. Перейдите на [my.telegram.org/apps](https://my.telegram.org/apps)
2. Войдите с вашим номером телефона
3. Создайте приложение
4. Сохраните API_ID и API_HASH

## 4. Настройка конфигурации

Создайте файл `.env`:

```bash
cp .env.example .env
```

Заполните `.env`:

```
BOT_TOKEN=ваш_токен
API_ID=ваш_api_id
API_HASH=ваш_api_hash
DB_HOST=localhost
DB_USER=root
DB_PASS=ваш_пароль
DB_NAME=portals_bot
```

## 5. Запуск бота

```bash
python bot.py
```

## 6. Первое использование

1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Отправьте `/add` чтобы добавить подарок
4. Введите название и модель подарка

## Команды

- `/start` - Список команд
- `/add` - Добавить подарок
- `/list` - Список отслеживаемых подарков
- `/stats` - Статистика

## Что дальше?

Бот автоматически:
- ✅ Проверяет цены каждую минуту
- ✅ Отправляет уведомления при снижении цен
- ✅ Сохраняет данные в MySQL базу

## Устранение проблем

**Бот не запускается:**
- Проверьте, что все переменные в `.env` заполнены
- Убедитесь, что MySQL сервер запущен
- Проверьте подключение к базе данных

**Ошибка аутентификации:**
- Проверьте правильность API_ID и API_HASH
- Убедитесь, что они получены с my.telegram.org/apps

**Не приходят уведомления:**
- Проверьте, что подарок добавлен через `/add`
- Убедитесь, что бот может отправлять вам сообщения (не заблокирован)
