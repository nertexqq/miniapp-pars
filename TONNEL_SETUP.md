# Настройка Tonnel Marketplace

## Получение TONNEL_AUTH

1. Откройте браузер и перейдите на https://market.tonnel.network
2. Войдите в свой аккаунт
3. Откройте консоль разработчика (F12 или Ctrl+Shift+I)
4. Перейдите на вкладку **Application** (или **Приложение**)
5. В левом меню выберите **Storage** → **Local Storage** → `https://market.tonnel.network`
6. Найдите ключ `web-initData`
7. Скопируйте **всё значение** этого ключа
8. Добавьте в файл `.env`:
   ```
   TONNEL_AUTH=ваше_значение_web-initData
   ```

## Использование

1. Установите библиотеку: `pip install tonnelmp`
2. Настройте `TONNEL_AUTH` в `.env` или `config.py`
3. Используйте команду `/settings` для выбора маркетплейса (Portals или Tonnel)
4. Бот будет работать с выбранным маркетплейсом

## Команды

- `/settings` - выбор маркетплейса (Portals/Tonnel)
- `/add` - добавить подарок для отслеживания (использует выбранный маркетплейс)
- `/get` - включить уведомления о новых подарках
- `/stop` - отключить уведомления о новых подарках
