# Portals Gifts GUI

Веб-интерфейс для мониторинга подарков с маркетплейсов Portals, Tonnel и MRKT.

## Установка

1. Установите зависимости:
```bash
pip install -r ../requirements.txt
```

2. Убедитесь, что в `.env` файле настроены:
   - `API_ID` и `API_HASH` (для Portals)
   - `PORTALS_AUTH` (опционально, токен Portals)
   - `TONNEL_AUTH` (токен Tonnel)
   - `MRKT_AUTH` (токен MRKT)

## Запуск

```bash
python gui/server.py
```

Откройте браузер и перейдите на http://localhost:5000

## Возможности

- ✅ Включение/выключение мониторинга
- ✅ Фильтры по маркетплейсам (Portals, Tonnel, MRKT)
- ✅ Фильтр по названию коллекции
- ✅ Фильтр по модели
- ✅ Фильтр по цене (мин/макс)
- ✅ Сортировка (последние, цена по возрастанию/убыванию)
- ✅ Отображение подарков в виде карточек с изображениями
- ✅ Real-time обновления через WebSocket
- ✅ Автоматическое обновление каждые 5 секунд

## Использование

1. Включите мониторинг переключателем вверху
2. Настройте фильтры (маркетплейсы, коллекция, модель, цена)
3. Нажмите "Применить фильтры"
4. Новые подарки будут появляться автоматически в виде карточек с изображениями
